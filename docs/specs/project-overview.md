# Project Overview

## Purpose

Дипломный проект — веб-приложение, разрабатываемое в рамках учебной практики УП.02.

## Tech Stack

- **Frontend:** Nuxt 4, Vue 3, TypeScript, Vite, SSR
- **Styling:** Vanilla CSS, БЭМ (BEM), CSS custom properties

## Задачи практики

1. **Описание тематики дипломного проекта** — 1–2 стр. А4: актуальность, ЦА, функционал, технологии, результат.
2. **Use-case диаграмма** — акторы и их взаимодействие с системой.
3. **Проектирование и разработка интерфейса** — минимум 3 страницы/экрана, HTML+CSS+TS.

## Acceptance Criteria

- [ ] Задание 1: описание проекта оформлено в `.docx`/`.txt`/`.pdf`/`.md`
- [ ] Задание 2: use-case диаграмма в виде изображения
- [ ] Задание 3: минимум 3 страницы, семантическая разметка, адаптивность, клиентская логика, ссылка на репозиторий
