# Concept: Anime Catalog

> Учебная практика УП.02. Черновик идей.

## Концепция

Сайт для просмотра аниме. Не энциклопедия (не Shikimori), а именно стриминговый сайт — каталог + плеер-заглушка.
Без бекенда. Данные из Shikimori API. Хранение пользовательских списков — localStorage.

## Стек

Nuxt 4 + Vue 3 + TypeScript + SSR, PrimeVue (UI-библиотека), Vanilla CSS (БЭМ)

## Срок

1 день.

---

## Название сайта

**AnimeBaza** — «База аниме». Коротко, запоминается, отражает суть.

---

## Визуальное направление: «Anime Night Market»

**Атмосфера:** ночной кинотеатр / аниме-конвент — тёмный фон, неоновые акценты, кинематографичная глубина.

**Цветовая палитра:**
- База: глубокий тёмный (`#0f0f1a`) — как зал кинотеатра перед сеансом
- Поверхности: три уровня глубины (фон страницы → карточка → всплывающий элемент)
- Неон-акценты: циан (интерактив), маджента (рейтинг/избранное), янтарь (ожидание в списках)
- Цвета статусов списков: синий (запланировано), зелёный (смотрю), бирюзовый (просмотрено), янтарь (отложено), красный (брошено)

**Ощущение:** кинематографичный, но живой — как витрина с постерами в ночном городе.

## Страницы

| # | Страница | Маршрут | Что на странице |
|---|----------|---------|----------------|
| 1 | **Каталог** | `/` | Сетка карточек, поиск по названию, фильтры |
| 2 | **Страница тайтла** | `/anime/[id]` | Постер, информация, плеер-заглушка, кнопка «В список», оценка |
| 3 | **Вход** | `/login` | Форма входа (любой логин → профиль) |
| 4 | **Профиль** | `/profile` | Имя пользователя, 5 списков аниме, кнопка выхода |

---

## Навигация (шапка сайта)

- **Название сайта** — текстом (без логотипа), клик → на главную
- **Каталог** — ссылка на `/`
- **Поиск** — строка поиска в шапке, динамический поиск (на лету, без Enter)
- **Профиль / Войти** — если авторизован: имя пользователя; если нет: кнопка «Войти»

**Кнопка «Выйти»** — в профиле.

---

## Каталог (главная страница)

**Первая загрузка:** показываем онгоинги (сейчас выходят).
**Подгрузка:** бесконечная лента (скролл → ещё тайтлы).

**Фильтры:**

| Фильтр | Как выбирать |
|--------|-------------|
| **По типу** | TV / Movie / OVA / ONA |
| **По статусу** | Онгоинг / Вышел / Анонс |
| **По году** | Выбор года |
| **По сезону** | Выбор сезона (зима/весна/лето/осень + год) |
| **По минимальному рейтингу** | От N баллов |
| **Сортировка** | По дате / Популярности / Рейтингу |

---

## Карточка в каталоге

- Постер + название + тип (TV, Movie, OVA, ONA)
- При наведении — попап сбоку с подробной информацией:
  - Год, жанры, описание, статус (вышел/онгоинг), длительность серии, кол-во эпизодов
  - Кнопка добавления в список (с выбором одного из 5 статусов)
  - Индикатор — добавлено или нет
- На мобильных: вместо hover-попада — bottom sheet по тапу

---

## Списки пользователя

Пять статусов. Хранятся в браузере (localStorage).

| Статус | Смысл | Цвет |
|--------|-------|------|
| Запланировано | Хочу посмотреть | Синий |
| Смотрю | Смотрю сейчас | Зелёный |
| Просмотрено | Посмотрел | Бирюзовый |
| Отложено | Поставил на паузу | Янтарь |
| Брошено | Не понравилось, бросил | Красный |

Если аниме удалили из каталога — ID само очистится из списка.

---

## UX и микровзаимодействия (идеи)

- Карточка при наведении чуть приподнимается и светится — будто берёшь коробку с диском
- Попап выезжает сбоку с лёгкой анимацией — не резко, а как выдвижной ящик
- Переключение табов в профиле — плавная смена контента
- Добавление в список — кнопка сжимается в точку и превращается в галочку
- Оценка — звёзды увеличиваются при наведении
- Скелетоны при загрузке — карточки-заглушки той же формы, что и настоящие

Все анимации — только CSS, без лишних JS-библиотек.

---

## Состояния интерфейса (идеи)

- **Загрузка:** карточки-скелетоны (повторяют форму постера + текста)
- **Пустой список:** не просто «нет», а иконка + сообщение + кнопка «В каталог»
- **Ошибка API:** сообщение + кнопка «Повторить»
- **Нет результатов поиска:** предложение попробовать другой запрос

Каждое состояние — не послевкусие, а часть дизайна.

---

## Плеер-заглушка

На странице тайтла — блок с постером аниме и кнопкой Play поверх него.
Надпись: «Видео временно недоступно». Без функционального плеера.

---

## Что точно НЕ делаем

- Бекенд (кроме прокси к Shikimori)
- Страницу избранного (заменено списками в профиле)
- Вкладки с персонажами и скриншотами
- Реальную регистрацию с валидацией
- Функциональный плеер
- Стиль под Shikimori/энциклопедию
